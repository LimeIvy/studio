# **App Name**: CourseFlow

## Core Features:

- Markdown Display: Display the course materials in a clear, readable format using a Markdown renderer.
- Interactive Stage Map: Visualize the course as a directed graph, showing the stages and their connections.
- Progress Tracking: Allow users to mark stages as complete using a button at the end of the stage content. Completed stages are visually distinguished on the stage map.

## Style Guidelines:

- Primary color: Dark blue (#1A237E) for a professional and trustworthy feel.
- Secondary color: Light gray (#F5F5F5) for backgrounds and subtle contrasts.
- Accent: Teal (#00ACC1) for interactive elements and progress indicators.
- Clean and modern sans-serif fonts for headers and body text to ensure readability.
- Simple and consistent icons for navigation and stage completion status.
- Use a clear, modular layout with sufficient spacing to avoid clutter.
- Subtle animations for transitions and feedback on user interactions.
